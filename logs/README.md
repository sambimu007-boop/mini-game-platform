# 🎮 Mini-Game Platform

Plateforme de mini-jeux multijoueurs avec salons privés, audio intégré et interface d'administration.

## 🚀 Installation
```bash
npm install
cp .env.example .env
npm run create-admin
npm start
# Dossier des logs
